# Goを用いたWebアプリケーション

## 使用したデータ
データ：[日本郵便株式会社の住所の郵便番号データ](https://www.post.japanpost.jp/zipcode/dl/utf-zip.html)を使用

## 内容
- 都道府県一覧の表示  
- 都道府県別による市区町村一覧の表示  
- 郵便番号から住所の表示  
- おおよその住所から郵便番号の検索
